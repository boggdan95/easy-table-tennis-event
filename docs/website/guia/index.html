Guía de Usuario — ETTEM

Easy Table Tennis Event Manager — Versión 2.6

ETTEM es una aplicación para gestionar torneos de tenis de mesa de forma profesional. Funciona 100% offline — no necesita conexión a internet. Todos los datos se guardan localmente en tu computadora.

Qué puedes hacer con ETTEM

  • Fase de Grupos (Round Robin): Todos contra todos dentro del grupo
  • Fase Eliminatoria (Knockout): Llave directa hasta el campeón
  • KO Directo: Bracket sin fase de grupos
  • Equipos: Swaythling, Corbillon y Olímpico
  • Scheduler: Asignación de partidos a mesas y horarios
  • Marcador móvil: Árbitros registran desde el celular
  • Pantalla pública: Resultados en vivo para TV
  • Branding: Logo, colores y datos del organizador
  • Exportar: Excel (6 hojas) y CSV combinado
  • Certificados: Diplomas personalizados para premiación
