Exportar Datos Nuevo V2.6

Descarga todos los datos del torneo en formato Excel o CSV para análisis, archivo o integración con otros sistemas.

Descripción General

A partir de la versión 2.6, ETTEM permite exportar todos los datos del torneo con un solo clic. Los datos exportados incluyen jugadores, grupos, partidos, standings y resultados del bracket — todo lo necesario para archivar el torneo, compartirlo con la federación o procesarlo en Excel o cualquier herramienta externa.

Nueva funcionalidad en V2.6 La exportación de datos es completamente nueva en la versión 2.6. Anteriormente, los datos del torneo solo eran accesibles a través de la base de datos SQLite interna.

Cómo Acceder a las Exportaciones

Puedes iniciar una exportación desde dos lugares:

  • Centro de Impresión: haz clic en el botón Exportar en la sección correspondiente.
  • Menú de administración: accede directamente a /admin/export/ para ver todas las opciones de exportación disponibles.

Formatos de Exportación

Excel (.xlsx) Nuevo V2.6

El archivo Excel generado contiene 6 hojas de cálculo, cada una con un conjunto de datos diferente del torneo:

Hoja Contenido
Jugadores Lista completa de todos los jugadores inscritos: nombre, apellido, género, país, ranking y categoría.
Grupos Todos los grupos creados por categoría: número de grupo, jugadores asignados y fase.
Partidos de Grupo Cada partido de la fase de grupos con los jugadores, marcador por sets y resultado final.
Standings Clasificación final de la fase de grupos: posición, jugador, partidos jugados/ganados/perdidos, sets y puntos.
Bracket Estructura completa del cuadro eliminatorio: rondas, enfrentamientos y jugadores en cada posición.
Resultados de Bracket Todos los partidos del bracket KO con marcadores por sets, ganadores y ronda correspondiente.

El archivo Excel se descarga directamente desde el navegador como torneo_[nombre]_[fecha].xlsx. Puedes abrirlo con Microsoft Excel, LibreOffice Calc, Google Sheets o cualquier aplicación compatible con el formato .xlsx.

CSV Combinado Nuevo V2.6

La opción de CSV combinado genera un único archivo de texto con todos los datos del torneo en formato CSV (valores separados por coma). A diferencia del Excel, todos los datos se incluyen en un solo archivo con una columna adicional que indica el tipo de registro.

Este formato es ideal para:

  • Importar los datos a bases de datos externas o sistemas de ranking de federación.
  • Procesamiento automático con scripts o herramientas de línea de comandos.
  • Entornos donde Excel no está disponible.
  • Archivo de larga duración en un formato abierto y universal.
¿Cuándo usar Excel y cuándo CSV? Usa Excel cuando necesites revisar o presentar los datos de forma organizada (páginas separadas, formato, sumas automáticas). Usa CSV cuando necesites procesar los datos con otro programa o importarlos a un sistema externo.

Exportación por Categoría

Puedes exportar los datos de todo el torneo o filtrar por categoría específica. Al seleccionar una categoría, el archivo exportado contendrá únicamente los datos correspondientes a esa categoría (jugadores inscritos en ella, sus grupos, partidos y resultados).

Notas Importantes

Los datos se exportan tal como están La exportación refleja el estado actual del torneo. Si hay partidos pendientes o resultados sin ingresar, esas celdas aparecerán vacías en el archivo exportado. Para obtener una exportación completa, realiza la exportación al finalizar el torneo con todos los resultados ingresados.
Sin conexión a internet La exportación funciona completamente de forma local. El archivo se genera en la computadora que ejecuta ETTEM y se descarga directamente en el navegador, sin necesidad de enviar ningún dato a servidores externos.
