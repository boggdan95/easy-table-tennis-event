Instalación y Primer Inicio

Cómo instalar ETTEM en Windows y macOS, y activar tu licencia.

Windows (Ejecutable)

  1. Descarga el archivo ETTEM.exe
  2. Colócalo en una carpeta de tu preferencia (ej: C:\ETTEM\)
  3. Haz doble clic en ETTEM.exe
  4. Tu navegador se abrirá automáticamente en http://127.0.0.1:8000
⚠ Advertencia de Windows SmartScreen

Al ejecutar ETTEM por primera vez, Windows mostrará una advertencia de seguridad porque la aplicación aún no cuenta con un certificado de desarrollador. Esto es normal y seguro.

Sigue estos pasos para continuar:

  1. Aparece la ventana “Windows protegió tu PC”
  2. Haz clic en “Más información” (texto azul debajo del mensaje)
  3. Aparece el botón “Ejecutar de todos modos” — haz clic en él
  4. ETTEM iniciará normalmente. Solo necesitas hacer esto una vez.

🔒 ¿Por qué aparece esto? Windows SmartScreen advierte sobre aplicaciones descargadas de Internet que no tienen firma digital de un certificado de desarrollador de Microsoft. ETTEM es una aplicación segura — el código no está firmado aún, pero esto no afecta su funcionamiento.

macOS (Aplicación)

  1. Descarga el archivo ETTEM.dmg
  2. Abre el .dmg y arrastra ETTEM a la carpeta Aplicaciones
  3. Abre ETTEM desde Aplicaciones
  4. Tu navegador se abrirá automáticamente
⚠ Advertencia de macOS Gatekeeper

macOS bloquea aplicaciones que no están firmadas con un certificado de Apple Developer. Al abrir ETTEM por primera vez verás el mensaje “ETTEM no se puede abrir porque es de un desarrollador no identificado”.

Esto es normal y seguro. Sigue estos pasos:

Opción 1 — Clic derecho (recomendado):

  1. Ve a la carpeta Aplicaciones en Finder
  2. Haz clic derecho (o Control + clic) sobre ETTEM
  3. Selecciona “Abrir” en el menú contextual
  4. Aparece un diálogo — haz clic en “Abrir” para confirmar
  5. ETTEM iniciará normalmente. Solo necesitas hacer esto una vez — las siguientes veces abre con doble clic normal.

Opción 2 — Desde Ajustes del Sistema:

  1. Intenta abrir ETTEM normalmente (doble clic) — aparece el bloqueo
  2. Ve a Ajustes del Sistema → Privacidad y Seguridad
  3. Busca el mensaje “Se bloqueó ETTEM” cerca del final
  4. Haz clic en “Abrir de todos modos”
  5. Confirma con tu contraseña o Touch ID

Opción 3 — Terminal (avanzado):

Si las opciones anteriores no funcionan, abre Terminal y ejecuta:

xattr -cr /Applications/ETTEM.app

Luego abre ETTEM normalmente con doble clic.

🔒 ¿Por qué aparece esto? macOS Gatekeeper bloquea aplicaciones que no tienen firma de Apple Developer ID. ETTEM es una aplicación segura — próximamente se incluirá la firma digital para eliminar esta advertencia.

Dónde se guardan los datos

PlataformaUbicación
Windows.ettem/ junto al ETTEM.exe
macOS~/Library/Application Support/ETTEM/

La carpeta contiene:

  • ettem.sqlite — Base de datos con todos los torneos
  • license.key — Tu clave de licencia
  • uploads/ — Logos y archivos subidos

Activar Licencia

La primera vez que abras ETTEM, verás la pantalla de activación.

  1. Ingresa tu clave de licencia en formato: ETTEM-XXXX-MMYY-SSSSSSSS
  2. Haz clic en Activar
  3. Si la clave es válida, accederás al panel principal

Estado de la Licencia

En la barra lateral izquierda verás los días restantes:

  • Verde: Más de 30 días restantes
  • Amarillo/Parpadeante: Menos de 7 días — renueva pronto
  • Expirada: Contacta al proveedor para renovar
